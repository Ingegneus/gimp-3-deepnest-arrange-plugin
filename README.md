this plugin takes an svg from deepnest and arranges the corresponding raster images in gimp

## what is deepnest?
[deepnest](https://github.com/deepnest-next/deepnest-next) is a nesting application. it takes svgs and nests/packs them tightly onto a sheet. with some preparation this plugin can take the generated svg and position raster images according to the deepnest svg.
